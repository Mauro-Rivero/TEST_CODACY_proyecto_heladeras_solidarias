# Flujo de Ejecución para Cargas de Oferta

1. **Selección de una Carga de Oferta**
   - Un Colaborador (Jurídico) selecciona una Carga de Oferta.

2. **Ejecución de una Carga de Oferta**
   - El Colaborador da de alta la Oferta.

3. **Confirmación de la Carga de Oferta**
   - El Colaborador confirma la exitosa ejecución de la Carga de Oferta.
      - (Las Cargas de Oferta no suman puntos).
